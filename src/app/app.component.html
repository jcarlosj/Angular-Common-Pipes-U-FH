UpperCasePipe & LowerCasePipe

# Entrada Pipe Salida
1 {{ name }} uppercase {{ name | uppercase }}
lowercase {{ name | lowercase }}

Descripción: Transforma una cadena de texto a mayusculas y/o minusculas

ngFor & slice:2:5

DecimalPipe

# Entrada Pipe Salida
1 {{ PI }} number {{ PI | number }}
number:'3.0-0' {{ PI | number:'3.0-0' }}
number:'1.0-2' {{ PI | number:'1.0-2' }}

Descripción: Transforma un número en una cadena, formateada de acuerdo con las reglas de configuración regional que determinan el tamaño y el separador del grupo, el carácter de punto decimal y otras configuraciones específicas de la configuración regional. El formato que debe usar este pipe es: minIntegerDigits.minFractionDigits-maxFractionDigits, por ejemplo: number:3.0-0 (Tres decimales, Cero como Minimo de dígitos decimales, Cero como Máximo de dígitos decimales)
