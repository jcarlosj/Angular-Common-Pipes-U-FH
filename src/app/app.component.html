DecimalPipe

# Entrada Pipe Salida
1 {{ PI }} number {{ PI | number }}
number:'3.0-0' {{ PI | number:'3.0-0' }}
number:'1.0-2' {{ PI | number:'1.0-2' }}

Descripción: Transforma un número en una cadena, formateada de acuerdo con las reglas de configuración regional que determinan el tamaño y el separador del grupo, el carácter de punto decimal y otras configuraciones específicas de la configuración regional. El formato que debe usar este pipe es: minIntegerDigits.minFractionDigits-maxFractionDigits, por ejemplo: number:3.0-0 (Tres decimales, Cero como Minimo de dígitos decimales, Cero como Máximo de dígitos decimales)

PercentPipe

# Entrada Pipe Salida
1 {{ sells }} percent {{ sells | percent }}
percent: '2.2-4' {{ sells | percent: '2.2-4' }}
percent:'2.3-5' {{ sells | percent:'2.3-5' }}

Descripción: Transforma un número en una cadena de porcentaje, formateada de acuerdo con las reglas de configuración regional que determinan el tamaño y el separador del grupo, el carácter de punto decimal y otras configuraciones específicas de la configuración regional.

En algunos casos requiere un código de configuración regional para las reglas de formato de configuración regional a utilizar. Cuando no se proporciona, utiliza el valor de LOCALE_ID, que es en-EE. UU. De forma predeterminada. Consulte Configuración de la configuración regional de su aplicación. Opcional. El valor predeterminado es indefinido

CurrentPipe

# Entrada Pipe Salida
1 {{ price }} currency {{ price | currency }}
currency: 'EUR' {{ price | currency: 'EUR' }}
currency: 'EUR':false {{ price | currency: 'EUR' : false }}
currency: 'CAD':'symbol':'4.0-0' {{ price | currency: 'CAD' : 'symbol' : '4.0-0' }}
currency:'INR':'Indian Rupee' {{ price | currency:'INR':'Indian Rupee' }}

Descripción: número que se formateará como moneda. Requeriraá algunos parámetros

El código de moneda ISO 4217, como USD para el dólar estadounidense y EUR para el euro. El código de moneda predeterminado se puede configurar utilizando el token de inyección DEFAULT_CURRENCY_CODE. Opcional. El valor predeterminado es indefinido. Formatos

El formato para el indicador de moneda. Uno de los siguientes: code, symbol(default), symbol-narrow, String, Boolean.

Opciones de representación decimal, especificadas por una cadena.
